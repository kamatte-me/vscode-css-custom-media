# VSCode Extension - CSS Custom Media Query Support

Extension for [CSS Custom Media Query](https://www.w3.org/TR/mediaqueries-5/#custom-mq)

## Features

- `@custom-media` syntax
- Autocomplete
- Go to definition
- Go to references
- Diagnostics
- Support file extensions: `.css`, `.scss`, `pcss` and `.postcss`
