# VSCode Extension - CSS Custom Media Query Support

Extension for [CSS Custom Media Query](https://www.w3.org/TR/mediaqueries-5/#custom-mq), which supports

- `@custom-media` syntax
- Autocomplete
- Go to definition
- Go to references
- Diagnostics
